# Photon Random Walk in a Star (Turtle)

Visual simulation of photons random-walking inside a star. Step lengths are sampled from an exponential distribution using the local mean free path.

## Requirements
- Python 3.x (standard library only)

## Run (Terminal)
From this folder:

python3 random_walk.py

## Controls
- Click the turtle window to exit.

## Notes
turtle behaves best when run as a script from Terminal.
